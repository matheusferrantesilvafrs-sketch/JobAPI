JobAPI

API REST desenvolvida em ASP.NET Core Minimal API para gerenciamento de candidaturas de emprego.

#Tecnologias

- ASP.NET Core Minimal API
- C#
- Entity Framework Core
- MySQL
- Swagger

Funcionalidades

- Criar candidatura
- Listar candidaturas
- Buscar candidatura por ID
- Atualizar candidatura
- Excluir candidatura
